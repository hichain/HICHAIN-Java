## 操作方法
- 場内
  - **左クリック**: カードを持つ(ホールド)
- 盤内
  - **ドラッグ**: 盤を移動 (ホールド中でも可能)
  
- ホールド中
  - **左クリック**: カードを置く: 左クリック
  - **右クリック**: キャンセル
  - **ホイールクリック/スペースキー**: カードの回転
  - **Cキー**: 盤を中心に戻す
  
- **DELETEキー**: リセット
